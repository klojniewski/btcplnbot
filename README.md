# btcplnbot

Simple trading bot to BitBay.net

## System requirements

- node.js > 7
- npm > 4
- MongoDB

## Installation

1. Rename `config/env.js-example` to `config/env.js`.
2. Create empty MongoDB database.
3. Change API keys, Exchange commision value and database link in `config/env.jg` file.
4. Run `npm install`.

## Running

`npm start`
